# Getting Start
1,move to project
```
cd docker_blueprint
```
2,build a image
```
$ docker-compose build
```
3,start
```
$ docker-compose up -d
```
4,login
```
$ docker-compose exec web /bin/bash
```
5, install from package.json
```
$ npm i
```
6, start gulp task
```
$ gulp
```

7, access to http://localhost:3000/
